# Bubble Sort

O Bubble Sort funciona comparando pares de elementos adjacentes e trocando-os de lugar se estiverem na ordem errada. Os maiores elementos "flutuam" para o final do array a cada iteração, como bolhas de ar subindo na água.

## Complexidade

| Melhor caso| Caso médio | Pior caso |
| :---: | :---: | :---: |
| O(n) | O(n²) | O(n²) |
