# Insertion Sort

Funciona de forma semelhante a como organizamos cartas de baralho na mão. Você pega um elemento e o "insere" na posição correta entre os elementos que já foram organizados.

## Complexidade

| Melhor caso| Caso médio | Pior caso |
| :---: | :---: | :---: |
| O(n) | O(n²) | O(n²) |
