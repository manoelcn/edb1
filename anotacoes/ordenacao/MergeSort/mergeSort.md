# Merge Sort

Divide o array recursivamente ao meio até que cada sub-array tenha apenas um elemento. Depois, ele "funde" (merge) esses sub-arrays de volta, ordenando-os durante a fusão.

## Complexidade

| Melhor caso| Caso médio | Pior caso |
| :---: | :---: | :---: |
| O(n log n) | O(n log n) | O(n log n) |
